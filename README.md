# ⚡ Haské Énergie - Mini-centrale Solaire Intelligente

Système de monitoring intelligent pour mini-centrale solaire avec IA.

## 🏗️ Structure

```
haske-energie/
├── ai/              # Intelligence Artificielle (Python Flask)
├── backend/         # Backend API (Node.js + Firebase)
└── frontend/        # Interface Web (React)
```

## 🚀 Installation

### Backend

```bash
cd haske-backend-firebase
npm install
npm start
```

### Frontend

```bash
cd haske-frontend
npm install
npm start
```

### IA

```bash
cd ai/api
pip install -r requirements.txt
python app.py
```

## 🔗 Ports

- Frontend: <http://localhost:3000>
- Backend: <http://localhost:5000>
- IA: <http://localhost:5001>
